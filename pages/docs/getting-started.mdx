# Getting started

**retro-avatar** create indenticon-like visual hashes styled like Github and Gravatar (retro) avatars

## Installation

To use **retro-avatar** in your project, you can install it as a dependency:

<Tabs items={['npm', 'yarn', 'pnpm']}>
  <Tab>
    ```bash copy
    npm install retro-avatar
    ```
  </Tab>
  <Tab>
    ```bash copy
    yarn add retro-avatar
    ```
  </Tab>
  <Tab>
    ```bash copy
    pnpm add retro-avatar
    ```
  </Tab>
</Tabs>
